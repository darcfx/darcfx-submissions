Title...........: ThinMailer
Author..........: Sean Dowd
Needs...........: a Unix server to run, MS WordPad to edit
About...........: POP3Client module >> acts as an interface to POP3 servers. A good benefiet about his program is that it's free, and you can edit the source code as much as you want because it's marketed "Public Domain." I guess the only bad thing is that you CANNOT view OR send file attachments, but that'll keep your server's speed up because it doesn't take away from the whole world updating through you. Uses the sendmail installed on the Unix servers. This is the same thing used on Darc FX.com, except I edited it.
