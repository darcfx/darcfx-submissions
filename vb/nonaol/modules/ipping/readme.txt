	      IP_Ping Module for Visual Basic 6.0
   	           (made in Visual Basic 6)
	           By Bradley Liang aka HawK
	         Instructions inside the Module

    This Module lets you 'ping' a server and return the
     response time.  This is useful for chat utilities and such.
     Originally made for my Winsock Chat Utility.

***Doesn't require any custom controls, but must have winsock dll***
