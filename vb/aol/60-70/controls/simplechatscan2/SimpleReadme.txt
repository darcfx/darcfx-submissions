Features:
Chat Scan
IM Scan
Chat Send
IM Send

Chat Scan and Chat Send are pretty self explanatory so we won't explain those.
IM Scan:
This event works like the Chat Scan except it alos returns the handle of the current IM
window that is found. 
IM Send:
Will send a message to any im window alredy open or a specific IM window if you pass the
handle throught the IMHandle variable. If no IMHandle is passed then the message will be
sent to the first IM window found.
