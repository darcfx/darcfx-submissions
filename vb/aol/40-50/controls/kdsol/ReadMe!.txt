StayOnline v1.0.1 ActiveX Control by KDesigns Inc. 1999

Created on Visual Basic 5.0

This ActiveX Control offers an easy element to your
AOL/VB application(s) to disable the 46 Minute Timer
and Idle Modal. It is freeware and yours to use at
your own will and risk.


To Raise the StayOnline Event, just change the prop for
Enabled in a Command button, a Sub, Form Load, whatever ya
want to use to start the process. 

Use either below:

Default is set to True/On

Turn On:     StayOnline1.Enabled = True
Turn Off:    StayOnline1.Enabled = False
