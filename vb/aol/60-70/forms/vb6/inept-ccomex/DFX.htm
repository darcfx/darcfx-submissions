Title...........: CCom Example
Author..........: Inept
Needs...........: Visual Basic 6.0
About...........: this is a small chat command example that i created for a few friends that were having trouble making a chat command tool. the forms were created in visual basic 6.0 for aols 4 and 5. i used dos chat.ocx and modCRUE.bas. modCRue.bas is included, but dos's chat ocx is not. this project is fully commented. from top to bottom you will find comentation. the people that i created this project for picked up the chat command concept in minutes. this should teach anyone the basics of chat commands with no problems at all. enjoy -inept
