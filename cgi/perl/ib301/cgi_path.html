>>CGI PATH Information
The CGI PATH is the 'server' path to the installer.cgi file.
Please note that this is not a URL. A PATH would look like /home/www/public_html/ikonboard/ib3
If you are unsure of your correct path, please consult with your webhost as Ikonboard.com cannot work this out for you.
Win95/NT users: Please enter double backslashes for the path seperator (Example: c:\\home\\htdocs\\ikonboard\\ib3
