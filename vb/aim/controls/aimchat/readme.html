About
Almost a year ago I, Quirk, wrote the first AIM ChatScan Control. It was the best until recently. Several people have made ChatScan Controls that are better than mine. I have found a new method of getting the chat lines which allows the control to get everyline. This ChatScan is also only for AOL Instant Messenger 3.5+.
Commands
A major difference between my old controls and this one is that this control is purely a chatscan whereis my old one was a pretty much a module compiled into a control. Here are the commands you will need to know.

Command Description
ChatScan1.Enabled = True Actives the scanning of the chat room for changes.
ChatScan1.Enabled = False Deactivates the scanning of the chat room for changes.
Call ChatScan1.ChatSend("Text") Sends "Text" to the current chat room.
X = ChatScan1.ChatName Sets X to the name of the current chat room.
X = ChatScan1.UserSN Sets X to the user's screen name.
Events
The best part about this control is that is is easy to get the last screen name and messgae from the chat.
When double clicking on the control the code window with the following appears:

Private Sub ChatScan1_Changed(Who As String, What As String)

End Sub

The variable Who is the screen name of the person who last said something in the chat room, and What is the variable that holds what that person said. If a message is sent my the server (So and So has entered the room, So and So has left the room, etc) Who is set to "OnlineHost". Here is an example of how to greet people when they enter the room.

Private Sub ChatScan1_Changed(Who As String, What As String)
If InStr(1, What, " has entered the room.") and Who = "OnlineHost" Then
TheSN = Mid(What,1, InStr(1,What," has entered the room.") - 1)
ChatScan1.ChatSend "Welcome " & TheSN & " to chat room " & ChatScan1.ChatName
End If
End Sub

I have included the entire project for the above example, sorry but the example is for Visual Basic 6 only.
